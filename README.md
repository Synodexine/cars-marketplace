# PGSM WMS

## Before run
Prepare environment variables:
```shell
$ cp .env.example .env
```

Build docker images
```shell
$ make build
```


## Run app
After the setup you can run the application
```shell
$ make up
```


## Run command
```shell
$ make run command = [your_command]
```
